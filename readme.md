# Venue Booking System

Venue Booking System คือระบบจองห้อง สามารถจองห้องและดูรายละเอียดของแต่ละห้องได้

## วิธีการจองห้อง

 - กรอกเลขห้อง 101-110
 - ใส่หัวข้อ
 - เลือกวันเวลา เริ่มและจบ
 - กดปุ่ม Booking

## วิธีดูรายละเอียดของแต่ละห้อง

 - กรอกเลขห้อง 101-110
 - กดปุ่ม Check Detail

## Installation

```bash
npm install
node server.js
```
